# Netflix-Singapore-Clone-2025


A **Netflix-inspired web application** built with **HTML, CSS, and JavaScript**. This project replicates the basic layout, styling, and interactive features of the Netflix homepage, giving a hands-on experience in front-end web development.

## Features

* **Responsive layout:** Adaptable to different screen sizes using CSS flexbox and grid.
* **Navigation bar:** Includes a logo, menu items, and a search bar.
* **Movie/show carousel:** Horizontally scrollable sections showcasing movies and series.
* **Hover effects:** Smooth animations when hovering over movie cards.
* **Interactive buttons:** Play, add to list, and info buttons with simple JS functionality.
* **Clean design:** Styled closely to Netflix’s homepage using CSS.

## Technologies Used

* **HTML5** – Structure of the web pages
* **CSS3** – Styling and responsive design
* **JavaScript (ES6)** – Basic interactivity, hover effects, and dynamic content

## Project Structure

```
netflix-clone/
│
├── index.html        # Main page
├── style.css         # Styling for layout and components
└── script.js         # JavaScript for interactivity
```

## How to Run

1. Clone or download the repository.
2. Open `index.html` in your preferred web browser.
3. Explore the interactive Netflix homepage clone.
